Help

  • Vendor name - Make sure it is the vendor who sent the bill and not the person who is receiving the bill. We tried to write a HINT to help you out there right above the blank. This is an auto-complete blank so if we have seen the vendor before you can just pick it, if not you have to type it.
  • Vendor address - This is also an auto complete blank so if we have seen it before it should have been populated when you picked the vendor. If that is the case just verify the information is correct. If not you will have to fill in the information for all the the address fields.
  • Invoice/Bill# - Dont confuse this with the customer number. This should be a number that is unique to the bill.
  • Invoice/Bill Date - This is the date the bill was created
  • Amount Due - This should be the total for just this particular bill, it should not include pervious bills. Meaning just for the line items that are present on the bill.
